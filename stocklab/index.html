stocklab.crawler module¶

This module provides serveral utilities which is not mandatory for stocklab to excecute, but these are common patterns for crawlers in applications.

class stocklab.crawler.SpeedLimiter(request_func, max_speed, tick_period)[source]¶

Bases: object

A helper class to limit action (e.g. remote access) frequencies.

Parameters

  • request_func (function) – The function which performs the action.

  • max_speed (float) – The speed limit in actions per second for the 2 most recent actions.

  • tick_period (float) – The cooldown interval in seconds.

Returns

Whatever request_func returns.
